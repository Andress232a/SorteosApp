# Página Web de Sorteos - PremioClick

Esta es la página web pública que muestra todos los sorteos disponibles y permite a los usuarios chatear con el administrador durante los sorteos en vivo.

## Características

- ✅ Visualización de todos los sorteos de la aplicación
- ✅ Filtros por estado (Todos, Activos, Finalizados)
- ✅ Chat en tiempo real con Socket.io
- ✅ Diseño responsive y moderno
- ✅ Integración completa con la API del backend

## Acceso

La página web está disponible en: `http://localhost:3001` (o el puerto configurado en el backend)

## Uso

1. Los sorteos se muestran automáticamente al cargar la página
2. Puedes filtrar por estado usando los botones en la parte superior
3. Haz clic en cualquier sorteo para abrir su link (si está configurado)
4. Usa el botón "Chat" para abrir el chat en vivo
5. Escribe tu nombre para comenzar a chatear

## Notas para el Admin

- El chat detecta automáticamente si un mensaje viene del admin (busca "Admin" en el nombre del usuario)
- Los mensajes del admin aparecen en color morado a la derecha
- Los mensajes de usuarios aparecen en blanco a la izquierda
